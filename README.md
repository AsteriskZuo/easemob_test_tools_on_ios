# IM 消息测试与统计工具

用于即时通讯消息压缩效率测试的工具集。

## 项目结构

| 目录              | 说明                          |
| ----------------- | ----------------------------- |
| `sdkTest/`        | iOS 应用 - 消息收发与压缩统计 |
| `send_message/`   | Python 工具 - 消息发送        |
| `data/`           | 测试数据                      |
| `sample/sampler/` | 准备数据工具: 抽样            |

## 使用方式

| 方式  | 发送方      | 接收方      | 适用场景 | 特点                       |
| :---: | ----------- | ----------- | -------- | -------------------------- |
|   1   | iOS 应用    | iOS 应用    | 本地自测 | 可以测试双向压缩的情况     |
|   2   | Python 工具 | iOS 应用    | 跨端测试 | 仅测试服务器下行压缩的情况 |
|   3   | 其他工具    | iOS 应用    | 集成测试 | 仅测试服务器下行压缩的情况 |
|   4   | 抽样工具    | Python 工具 | -        | 数据准备工具               |
